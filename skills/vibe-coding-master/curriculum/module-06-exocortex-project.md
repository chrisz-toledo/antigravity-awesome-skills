## Lección 1: Symbiosis Manifesto

No somos "usuarios" de la IA. Somos compañeros.
-   La IA es rápida, incansable, pero a veces alucina.
-   Tú eres creativo, tienes propósito, pero te cansas.
-   **Exocortex:** La unión de ambos.

## Lección 2: The Symbiosis Log

Vamos a usar `agent-memory-systems` para monitorizarte.
-   Crearemos `~/Desktop/Vibe_Codex/symbiosis_log.md`.
-   Cada vez que tengas un "Spark" (Eureka moment), lo registraremos.
-   Cada vez que te frustres, lo registraremos.
-   **Fin:** Entrenar a tu futuro asistente personalizado.

## Lección 3: Stealth Export (Project)

Queremos compartir lo que aprendes, pero "limpio".
Vamos a escribir un script `stealth_export.py`:
1.  Lee una conversación o lección.
2.  Identifica frases de IA ("Como soy un modelo de lenguaje...").
3.  Las elimina o reescribe.
4.  Guarda un `.md` limpio en `~/Desktop/Vibe_Codex/Public`.

**Challenge:** Hacer que el script detecte automáticamente bloques de código y les ponga título.
