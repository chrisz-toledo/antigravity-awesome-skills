# Module 01: Foundations (Fundamentos de Vibe Coding)

## Objetivos
-   Configurar el entorno (Antigravity + Cursor).
-   Entender el rol del "Manager" (Tú diriges, la IA escribe).
-   **Quick Win:** Desplegar tu primera web en 10 minutos.

## Conceptos Clave (Technical English Protocol)

| Español | Inglés (Bridge) | Immersion Context |
| :--- | :--- | :--- |
| Entorno de Desarrollo | **IDE (Integrated Development Environment)** | "Open your **IDE** (Cursor)..." |
| Línea de Comandos | **Terminal / Shell** | "Run this command in your **Terminal**..." |
| Desplegar | **Deploy** | "Let's **deploy** your site..." |
| Repositorio | **Repository / Repo** | "Clone the **repo**..." |

## Lección 1: El Mindset (The Mindset)

El "Vibe Coding" no es copiar y pegar. Es **orquestar**.
Tú eres el Arquitecto. La IA es el Albañil incansable.
Tu trabajo es tener la visión clara y saber *pedir* (Prompting).

## Lección 2: Hello, World... But Cooler

Olvídate de imprimir "Hola Mundo" en negro sobre blanco.
Vamos a hacer una **Personal Landing Page**.

1.  **Prompting:** Pediremos a la IA una estructura HTML5 moderna con CSS (Tailwind idealmente).
2.  **Vibe Check:** ¿Se ve bien? No. Dile que la haga "Cyberpunk" o "Minimalist".
3.  **Code Review:** Leeremos el código. No necesitas saber escribir cada etiqueta `<div>`, pero necesitas saber qué *hace*.
    -   *Reto:* Encuentra dónde está el color de fondo y cámbialo manualmente.

## Lección 3: Deploy (¡En vivo!)

Usaremos un servicio simple (como Vercel o Netlify drop) o GitHub Pages.
-   **Aha Moment:** Ver tu página accesible desde tu teléfono en menos de 10 minutos.
