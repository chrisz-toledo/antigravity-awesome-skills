# Module 05: The Connected Brain (NotebookLM Integration)

## Objetivos
-   Usar NotebookLM como tu "Cerebro Secundario".
-   El Workflow de "Captura -> Triage -> Update".
-   Crear Sourcebooks.

## Conceptos Clave (Technical English Protocol)

| Español | Inglés (Bridge) | Immersion Context |
| :--- | :--- | :--- |
| Fuente | **Source** | "Add this PDF as a **source**..." |
| Consulta | **Query** | "Run a **query** against your notebook..." |
| Citación | **Citation** | "Check the **citation** for that claim..." |
| Base de Conocimiento | **Knowledge Base** | "Update your **knowledge base**..." |

## Lección 1: NotebookLM Setup

1.  Crear cuaderno "Vibe Coding Journey".
2.  Subir: `implementation_plan.md` (sí, el de este proyecto).
3.  Subir: Tus notas de las lecciones anteriores.

## Lección 2: The Workflow (Continuous Learning)

Cómo mantenerte actualizado sin volverte loco:

1.  **Capture:** Ves un video de YouTube interesante (como los que enviaste). Copias el link. Lo pegas en NotebookLM.
2.  **Triage:** Le preguntas a NotebookLM: "¿Qué conceptos nuevos hay aquí que no sepa?".
3.  **Update:** Si es valioso, lo añades a tu lista de "Aprender".

## Lección 3: Talking to Your Past Self

NotebookLM permite "hablar" con tus documentos.
-   *Uso Real:* Cuando tengas 50 scripts en tu `Vibe_Codex`, súbelos a un notebook "My Codebase".
-   *Query:* "¿Cómo hice para ordenar archivos en Python hace 3 meses?".
-   *Resultado:* Tu propio código explicado por la IA.
